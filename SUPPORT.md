# Support

For usage questions, bugs, and feature requests, use GitHub Issues:

https://github.com/SanQianX/Token-Consumption-Leaderboard/issues

Please include:

- Your operating system.
- Node.js version.
- Tokboard version from `tokboard -v`.
- The command you ran.
- Any terminal output that does not include private data.

Do not paste Claude Code logs, local project paths, secrets, customer data, or
other private information into public issues.
